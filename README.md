**Aether** is a dual-paradigm interactive cognitive space visualizer. It demonstrates seamless bijective design principles, uniting modern declarative rendering architecture with historic physical and synchronous computer engineering paradigms in a single, high-fidelity experience.

Whether charting AI-categorized thought paths using the standard **Jetpack Compose Physics canvas**, panning around a beautiful **Three.js WebGL Orbit cluster**, or running legacy **Y2K tests over synchronous SOAP v1.1 envelopes**, Aether is engineered to captivate and amuse minds across all generations of computer science.

---

## 🎨 Creative Philosophy: Bijective Unified State

The cornerstone of **Aether** is its strict adherence to **Bijective Design**:
*   Every high-dimensional virtual node has a physical counterpart.
*   Every touch/drag on the modern WebGL Canvas directly manipulates variables linked back to the Android ViewModel.
*   Every string typed into the modern core is mathematically rendered into physical holes on an output-ready **80-column Hollerith Punchcard** under the legacy mainframe simulation.

## 🔧 Installation & Native Setup

### Prerequisites
*   **Android Studio Koala or newer**
*   **Android SDK 24+** (Android 7.0 Nougat to Android 15)
*   **Gradle 8.0+**

### Building from Source

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/aether-quantum-engine.git
   cd aether-quantum-engine
   ```

2. Sync with Gradle and build the apk:
   ```bash
   # Run compile tasks
   ./gradlew assembleDebug
   ```

3. To record test screenshots via Roborazzi:
   ```bash
   ./gradlew recordRoborazziDebug
   ```
