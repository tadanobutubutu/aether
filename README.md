# 🌌 Aether

### Interactive Gravitational Personal Knowledge Mapping (PKM) for Android

Aether is a modern, cognitive personal knowledge management system that transforms raw ideas, notes, and task lists into an interactive gravitational solar system. By mapping the importance of thoughts to physical mass and categorizing them as orbiting planets and satellites, Aether turns traditional hierarchical files into a visual and tactile galaxy of wisdom.

## 🎨 Core Values & Human Experience

### 1. Gravitational Mindscapes
Ideas are not files in a folder; they represent a network of thoughts drawing from each other.
* **The Sun:** Represents crystallized wisdom and the center of the viewport system.
* **Planets:** Major cognitive domains/categories.
* **Satellites (Moons):** Actions and thoughts, orbiting corresponding planets based on importance scores.

### 2. Primary MVP Persona Focus
While Aether targets creative writers, engineers, and researchers, **Scenario B (The Advanced Software Engineer's Knowledge Pipeline)** is our MVP baseline focus. Our classification pipelines are specifically optimized to recognize software concepts and prioritize learning targets efficiently.

### 3. The Functional Retro Emulator Layer
The emulated vintage console is more than an aesthetic layout. It symbolizes the historical roots of computing (COBOL punch cards, SOAP XML envelopes, and Perl CGI).
* **Game Logic:** Frequent communication can trigger a `CGI Buffer Overflow` (25% chance), forcing you to navigate to the Mainframe terminal to patch buffer registers manually. This creates a fun, retro-tech debug experience alongside modern task tracking.
* **Double State Integrity:** All actions performed on the physical 2D canvas dynamically generate real-time log outputs on the retro mainframe logs screen, simulating old-school hardware feedback.

## 📸 Visual Assets & Screen Representations

Baseline visual outputs are integrated on the device and validated during build cycles.

* **Constellation View (2D Jetpack Compose Canvas):**
  * Displays orbits, gravity vectors, active satellite nodes, and real-time physical momentum drag.
* **Mainframe Console View (Virtual retro COBOL emulator):**
  * Houses interactive sliders to pan system sizes, buttons to patch CGI buffers, and active ASCII punch cards.
* **Galaxy View (WebGL 3D WebAssembly Canvas):**
  * Fluid webGL-rendered particles that orbit central matrices, managed on the JavaScript thread using compiled low-level Rust and Zig WASM coprocessors.
