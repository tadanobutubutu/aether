**Aether（エーテル）**は、あなたの混沌としたアイデア（流れる意識）をAIが認知分類し、それらを「太陽・惑星・衛星」の軌道を持つ美しい重力物理システムとして視覚化・操作する、まったく新しいアプローチの**AI駆動型パーソナル・ナレッジ・マネジメント（PKM：個人知識管理）システム**です。

複雑なフォルダ階層や、静的な2Dマインドマップから脱却しましょう。Aetherは「物理演算が宿る2Dキャンバス」と「瞑想的な3D全体俯瞰空間」、さらに「レトロな双方向エミュレータ」をシームレスに交差させ、あなたの中に眠るアイデア同士の引力と接続を再発見させます。

## 🎨 Core Value Propositions (どのような体験と価値を提供するのか)

### 1. 非線形な知識の連想マッピング (Gravitational Mindscapes)
アイデアは本来、ツリー構造ではなく、相互に引き合うネットワークです。Aetherでは、重要度（重要性）が「質量」となり、カテゴリが「太陽・惑星」、個別の考えが「衛星」となります。惑星同士の距離、衛星が描く物理シミュレーション軌道を通して、直感的かつ視覚的にアイデア同士の関連の強さを捉えられます。

### 2. インフェースの機能的二面性 (The Structural Dichotomy)
Aetherは、以下の2つの視点を相互に行き来することで、脳の活性化と情報の整理を最大化します。
*   **【2D Dynamics — 編集と干渉のアクション空間】** (Jetpack Compose Physics)  
    実時間で滑らかに動く物理エンジン上。直接のドラッグ＆ドロップ、軌道半径のストレッチ、重要度の拡大縮小、アイデアのゴミ箱への投棄、または他の惑星との引力吸収（カテゴリ自動再割当て）など、触覚を通して直感的な「アイデアの直接編集」を行います。
*   **【3D Reflection — 全体トポロジーの瞑想・観察空間】** (WebGL / Three.js via WebView)  
    作成された思考の銀河を3D空間に投影。シームレスな回転・ピッチ操作や自動追跡カメラ、美しい発光するクォンタムメッシュにより、アイデアを多角的に俯瞰して俯瞰し（Reflection）、新たなつながりの発見やアイデアの結合を刺激します。

## 👤 Real-World Use Case Scenarios (実利用シナリオ)

Aetherは様々なクリエイターや知識探求者に、ユニークで強力なインスピレーションを与えます。

### ✍️ シナリオ A: 小説家・世界観の創作者 (World Builders)
*   **用途:** 複雑な登場人物の相関関係やプロットテーマの可視化。
*   **活用例:** 各人物を「衛星」として作成。彼らが帰属する「惑星（勢力や派閥）」の周りに配置します。ドラッグ操作で勢力を変更すると、その間に流れる重力軌道が変わり、キャラクターの相関バランスを視覚的にダイナミックにシミュレーションできます。

### 💻 シナリオ B: プログラマー・技術習得のロードマップ
*   **用途:** 技術スタック、フレームワーク知識、そして日々の学習インサイトの集約。
*   **活用例:** 学習目標のテキストからAIが自動で「Core Language（言語）」「Database（DB）」「Security（セキュリティ）」といった惑星へとインサイトを振り分けます。重いインサイト（重要性が高い）ほど大きく、太陽に近い軌道をハイスピードで回り、あなたに学習の優先順位を常に促します。

### 📚 シナリオ C: 研究者・アイディエーター (Knowledge Explorers)
*   **用途:** 不揃いで未整理なアイデアの連想マッピング。
*   **活用例:** 朝のブレインストーミングで得たアイデア群を一気にアプリに話し込む（音声やフリーテキスト）。Geminiがそれらを瞬時に分解し、惑星の引力圏に再配置。3D空間全体をクルーズし、どのカテゴリにどれだけの思考密度が溜まっているか（銀河の疎密）を一目で把握します。

## 📟 The Fun Retro & Emulation Layer (遊び心あふれる古代技術ジョークとゲーム性)

Aetherは極めて真剣に、かつ細部にこだわって「双方向デザイン（Bijective Design）」を極限化するために、**100% Kotlin/JVMで駆動する完全ローカル・レトロエミュレータ**を搭載しています。
*   **技術的な透明性 (No Complex Toolchains):** 
    あなたが本アプリをビルドするにあたり、COBOLコンパイラ、Perlエンジン、外部SOAPサーバーの立ち上げは**一切必要ありません**。これらはすべてKotlinによる疑似シミュレータとして、Android SDK単体で完璧かつ高速に再現されています。
*   **ダイナミックなビジュアライザー:**  
    入力した思考テキスト、あるいは2D物理キャンバスでの指による移動アクションは、即座にレガシーメインフレーム用の変数へと数学的にマッピングされ、COBOLフォーマットのパンチカードへ物理的に穴（▇）が穿たれ、シミュレートされたPerl-CGIを介したSOAP v1.1 XMLエンベロープにシリアル化されてログに流れます。
*   **インタラクティブなデバッグ体験 (CGI Buffer Overflow):**  
    レガシー（PERL SOAP ROUTE）通信経由でAI結晶化を実行していると、**25%の確率でメインフレーム内のCOBOLレジスタバッファが一時的に溢れてエラー（CGI Buffer Overflow）がトリップするゲーム要素**が組み込まれています。
    エラー発生時は「メインフレーム・ターミナル」画面へとダイブし、手動で **[PATCH DRIFT & FLUSH CGI BUFFER]** を叩いてキャッシュを消去・リセットしなければ、再び結晶化が行えない、という「新旧テクノロジーが衝突するリアルなデバッグ感」を楽しめます。
*   **物理メカニカルビープ音:**  
    ダイヤル調節、パンチカードの刻み、バッファオーバーフローの警告時には、Androidの `ToneGenerator` を駆使した懐かしくもリアルな電子端末音（CDMA警告、チケット音など）が鳴り響きます。

## 🔧 Installation & Native Setup

### 📋 Prerequisites
*   **Android Studio Koala (or newer)**
*   **Android SDK 24+** (Android 7.0 Nougat to Android 15 ready)
*   **Gradle 8.0+**

### 🛠️ Building from Source
1. リポジトリをクローンします。
   ```bash
   git clone https://github.com/tadanobutubutu/aether.git
   cd aether
   ```
2. Android Studioを開き、「Import Project」を実行してシンクを完了します。
3. デバッグ版APKをビルドして実機/エミュレータに展開します。
   ```bash
   gradle assembleDebug
   ```
4. 単体テストや、Roborazziによるスクリーンショットテストを検証します。
   ```bash
   gradle verifyRoborazziDebug
   ```
